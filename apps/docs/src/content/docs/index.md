## What is Starbase?

Starbase is an open-source AI-native developer environment designed for repo setup, agent configuration, local memory, and repeatable coding workflows.

## Key Features

- **Hybrid Monorepo** — Turborepo for TypeScript packages plus Cargo workspace for Rust crates and CLI logic
- **AI-Native** — Built from the ground up for LLM-assisted development workflows
- **Local-First** — Keeps your data and agent configs on your machine
- **Repeatable** — Codified workflows you can run, share, and evolve
